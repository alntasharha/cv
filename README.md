## 🚀 Déploiement

### Option 1: Netlify (Recommandé)
1. Connectez-vous sur [netlify.com](https://netlify.com)
2. Glissez-déposez ce dossier
3. Le site est en ligne instantanément !

### Option 2: GitHub Pages
1. Créez un repository sur GitHub
2. Poussez ce code: `git push origin main`
3. Activez GitHub Pages dans Settings > Pages
4. Sélectionnez la branche `main`

### Option 3: Déploiement local
Ouvrez simplement `index.html` dans votre navigateur.
