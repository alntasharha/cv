# Déploiement sur GitHub Pages

## Étape 1: Créer un compte GitHub
1. Allez sur https://github.com
2. Cliquez sur "Sign up"
3. Utilisez votre email: albinamani25@gmail.com
4. Créez un nom d'utilisateur (ex: amani-albin ou albinamani)
5. Définissez un mot de passe
6. Vérifiez votre email

## Étape 2: Créer un repository
1. Connectez-vous à GitHub
2. Cliquez sur le bouton vert "New" ou "+" → "New repository"
3. Nom du repository: `amani-cv` ou `cv-portfolio`
4. Laissez "Public" sélectionné
5. **IMPORTANT**: Ne cochez PAS "Add a README file"
6. Ne cochez PAS "Add .gitignore"
7. Ne cochez PAS "Choose a license"
8. Cliquez sur "Create repository"

## Étape 3: Uploader les fichiers
Après création du repository, vous verrez des instructions. Cliquez sur:
"uploading an existing file"

Uploader ces fichiers depuis votre dossier:
- index.html
- styles.css
- script.js
- al.JPG (votre photo)

## Étape 4: Activer GitHub Pages
1. Dans votre repository, allez dans l'onglet "Settings"
2. Dans le menu de gauche, cliquez sur "Pages"
3. Source: sélectionnez "Deploy from a branch"
4. Branch: sélectionnez "main" et "/ (root)"
5. Cliquez sur "Save"

## Étape 5: Voir votre site
- Attendez 2-5 minutes
- GitHub vous donnera un lien: https://votre-username.github.io/amani-cv
- Partagez ce lien !
